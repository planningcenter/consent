# Person Header

> A header element for detailing, and acting on a person

## Basics

`person-header` is a CSS Grid with two grid areas: `avatar` and `detail`.

<Preview>
  <Story name="Basic">
    <div style={{ backgroundColor: "var(--product-sub-header--background-color)" }}>
      <div data-person-header--container>
        <img src={image} alt="kitten" data-person-header--avatar data-avatar />
        <div data-person-header--detail>
          <div data-person-header--name data-text>Kitty das Robot</div>
          <div><span data-text>kittydasrobot@email.com</span></div>
        </div>
      </div>
    </div>
  </Story>
</Preview>

| data-attribute                   | usage |
|----------------------------------|---------------|
| `data-person-header--container` | grid template |
| `data-person-header--avatar`    | treatment for `avatar` area and `data-avatar` element |
| `data-person-header--detail`    | treatement for `details` area |
| `data-person-header--name`      | treatment for `name` and `text` element |


## Inactive

When a person is inactive, the `avatar` is reduced to half opacity.

| data-attribute  | usage |
|-----------------|---------------|
| `data-person-header="inactive"` | treatment for `avatar` element for `inactive` people |
