# Utilities

## visually-hidden

The `visually-hidden` utility hides content in a way remains readable by assistive technologies. Use this when you need content to be read by assistive technologies but not shown on screen.

**Note**: This utility is designed for use exclusivily in `grids`. It should not be used directly elsewhere. Prefer duplication.

  <Story name="visually-hidden">
    <p>
      There ane two paragraphs here. This one can be seen on screen. The other

    </p>
    <p data-grids-visually-hidden>
