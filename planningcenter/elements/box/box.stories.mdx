## Sizing

Box can be changed from `border-box` (default) to `content-box`, using the custom property `--box--box-sizing`.

<Preview>
  <Story name="Sizing">
    <div data-box style={{ "--box--box-sizing": "content-box"}}>
      <p data-text style={{ backgroundColor: "white" }}>This box is <code>box-sizing: content-box</code>.</p>
    </div>
  </Story>
</Preview>

## Extension

There are no additional controls for `box`.  
However, it can be extended product-side or by importing common extensions.

The example below uses utility-class style extension for padding:

<Preview>
  <Story name="Extension">
    <div data-box="p:8 ph:4 pb:1" style={{ backgroundColor: "fuchsia"}}>
      <p style={{ backgroundColor: "white" }}>This box is <code>box-sizing: content-box</code>.</p>
    </div>
  </Story>
</Preview>
