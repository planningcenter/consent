# Box extension

## `p:{point}`

Padding in 8px increments.

<Preview>
  <Story name="p">
    <div data-box="p:8 ph:4 pb:1" style={{ backgroundColor: "fuchsia" }}>
      <p style={{ backgroundColor: "white" }}>
        This box is <code>box-sizing: content-box</code>.
      </p>
    </div>
  </Story>
</Preview>
