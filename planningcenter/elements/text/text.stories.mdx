## color

Text's color can be changed using the CSS Custom Property `--text--color`.  

<Preview>
  <Story name="color">
    <div style={{ display: "grid" }}>
      <span data-text style={{ "--text--color": "fuchsia" }}>
        This is <code>fuchsia</code>
      </span>
      <span data-text style={{ "--text--color": "tomato" }}>
        This is <code>tomato</code>
      </span>
    </div>
  </Story>
</Preview>

## font-size

Text's font-size can be changed using the CSS Custom Property `--text--font-size`.  
Here is an example of the available scale using that property:

<Preview>
  <Story name="font-size">
    <div style={{ display: "grid" }}>
      <span data-text style={{ "--text--font-size": ".625rem" }}>
        This is .625rem/10px
      </span>
      <span data-text style={{ "--text--font-size": ".75rem" }}>
        This is .75rem/12px
      </span>
      <span data-text style={{ "--text--font-size": ".875rem" }}>
        This is .875rem/14px
      </span>
      <span data-text style={{ "--text--font-size": "1rem" }}>
        This is 1rem/16px
      </span>
      <span data-text style={{ "--text--font-size": "1.125rem" }}>
        This is 1.125rem/18px
      </span>
    </div>
  </Story>
</Preview>
