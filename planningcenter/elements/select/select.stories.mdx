## disabled

Adding the `disabled` attribute to a `select` element gives it a disabled appearance.  


<Preview>
  <Story name="disabled">
    <select data-select disabled>
      <option>Taylor Swift</option>
      <option>Fearless</option>
      <option>Speak Now</option>
      <option>Red</option>
      <option>1989</option>
      <option>Reputation</option>
      <option>Lover</option>
    </select>
  </Story>
</Preview>

There are no custom properties for states.  
To style `:focus` and `:disabled` states, override the entire selector:  

```css
[data-select]:focus { /* ... */ }
[data-select]:disabled { /* ... */ }
```

They could also be written as extensions:

```css
[data-select="variant:bare"] { /* ... */ }
[data-select="variant:bare"]:focus { /* ... */ }
[data-select="variant:bare"]:disabled { /* ... */ }
```

These could also be provided product-specific classes or CSS-in-JS values.

```css
.alt-select { /* ... */ }
.alt-select:focus { /* ... */ }
.alt-select:disabled { /* ... */ }
```
