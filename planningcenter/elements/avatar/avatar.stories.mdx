# Avatar

> A circular image representing one person

## Custom Properties

Selector: `data-avatar`

Custom property names and defaults:

```css
--avatar--size: 2rem;
--avatar--border: 0 solid white;
--avatar--opacity: 1;
```

## Basic

<Preview>
  <Story name="Basic">
    <img data-avatar src={image} alt="kitten" />
  </Story>
</Preview>

## Size

Avatar's size can be changed using the CSS Custom Property `--avatar--size`.  
Here is an example scale using that property:

<Preview>
  <Story name="Size">
    <React.Fragment>
      <img
        data-avatar
        style={{ "--avatar--size": "1.25rem" }}
        src={image}
        alt="kitten"
      />
      <img
        data-avatar
        style={{ "--avatar--size": "1.5rem" }}
        src={image}
        alt="kitten"
      />
      <img
        data-avatar
        style={{ "--avatar--size": "3rem" }}
        src={image}
        alt="kitten"
      />
    </React.Fragment>
  </Story>
</Preview>

## Border

Avatar doesn't have a border or padding but it handle them.  
Here's an example of avatar being extended with a `status` indicating border â€” like Services:

<Preview>
  <Story name="Border">
    <img
      data-avatar
      style={{
        border: "calc(var(--avatar--size) * .1) solid hsl(7, 80%, 52%)",
      }}
      src={image}
      alt="kitten"
    />
  </Story>
</Preview>

`--avatar--size` is defined in the `[data-avatar]` scope. So it can be used in inline declarations for calculations.

_Note that absolute height is retained, using `box-sizing: border-box`_
