# Table

## Details

Selectors:

```
data-table
data-table-head
data-table-row
data-table-data
data-table-body
```

Custom properties and defaults:

```css
/* --tabel--border-color colors both outside border and horizontal row dividers */
--table--border-color: hsl(0, 0%, 90%);
/* root font-size controls relative scale of header and data elements */
--table--font-size: 0.875rem;
--table--border-radius: 0;
--table-row--height: 4rem;
--table-head--background-color: hsl(0, 0%, 97.5%);
--table-head--color: hsla(0, 0%, 40%);
--table-head--font-size: 0.75rem;
--table-data--border-color: var(--table--border-color);
--table-data--padding: 0 0.5rem;
```

## font-size

Table font-size cascades to all table cells and is controlleb by the custom property `--table-font-size`.  
Headings can additionally fine-tuned with the custom property `--table-head--font-size`.
